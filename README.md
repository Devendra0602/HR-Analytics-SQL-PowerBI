# HR Analytics Dashboard (SQL Server & Power BI)

This project focuses on analyzing employee attrition using SQL Server and Power BI.
The objective was to understand workforce trends and identify key factors contributing
to employee attrition through data-driven insights.

The project follows a simple and practical workflow, starting from raw data exploration
in SQL to building an interactive dashboard in Power BI.
